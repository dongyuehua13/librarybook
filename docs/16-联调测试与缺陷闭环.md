## 2. 联调范围

从用户端和管理端两边将系统连起来跑通，覆盖：

| 方向 | 覆盖点 |
|------|--------|
| 用户端 | 首页统计 → 座位列表 → 座位详情 → 预约提交 → 我的预约 |
| 管理端 | 管理员登录 → 座位管理 → 预约管理 → 统计面板 |
| 状态回流 | 管理端禁用座位 → 用户端不可见；预约数据两端一致 |
| 权限拦截 | AdminAuthFilter 未登录重定向、AJAX 401 |
| 关键规则 | 冲突检测、不可约禁用座位、非法取消、AdminAuthFilter 豁免 |

## 9. Bug 记录与闭环

### Bug-001: AdminAuthFilter 无限重定向循环

| 字段 | 值 |
|------|-----|
| 严重级别 | **P0（阻塞）** |
| 发现时间 | 2026-07-14 |
| 发现方式 | Playwright 自动点击：`/Admin/Login` 返回 `ERR_TOO_MANY_REDIRECTS` |
| 根因 | `AdminAuthFilter` 拦截 `Login` Action → 未登录重定向到 `Login` → 再次拦截 → 死循环 |
| 影响范围 | 全部管理端页面无法访问 |
| 修复方式 | `Filters/AdminAuthFilter.cs` 中增加 Action 名检查：`if (action == "Login" || action == "Logout") return;` |
| 修复验证 | Playwright 重新执行：P06/P07/P08/P09 全部通过 ✅ |
| 闭环状态 | **✅ 已修复已关闭** |
