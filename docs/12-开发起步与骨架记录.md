## 4. 数据库初始化方式

**Code First + EnsureCreated()**

Program.cs 启动时自动流程：
```
db.Database.EnsureCreated()    # 库不存在时创建库和所有表
DbInitializer.Seed(db)         # Users 表为空时写入种子数据
```
