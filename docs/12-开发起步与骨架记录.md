## 6. 数据库初始化方式

**Code First + EF Core Migrations**（与 docs/07 §7 的 EnsureCreated 策略不同，此处按本轮要求采用迁移模式）

首次启动流程：
```
Program.cs 启动
  └─ db.Database.Migrate()    # 自动应用全部迁移
  └─ DbInitializer.Seed(db)   # 自动插入种子数据（仅首次）
```

手动命令（如需）：
```bash
dotnet ef migrations add <名称>    # 创建新迁移
dotnet ef database update          # 应用迁移到数据库
```

## 8. 运行步骤

```bash
# 1. 检查环境
dotnet --version                              # 应为 10.x
dotnet ef --version                           # 应 >= 10.0

# 2. 还原 + 编译
cd src\LibrarySeatReservation.Web
dotnet restore
dotnet build

# 3. 迁移数据库（首次）
dotnet ef database update

# 4. 运行
dotnet run --urls "http://localhost:5002"

# 5. 浏览器打开
# http://localhost:5002
```
