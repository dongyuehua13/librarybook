## 1. 关键链路选择说明

### 1.1 链路认定

前序文档已明确两条主链路，本文直接沿用，不重新命名、不改步骤、不改页面职责。

| 链路 | 名称 | 来源文档 | 涉及页面 |
|------|------|---------|---------|
| 链路一 | 学生预约座位（闭环路径） | docs/04 §4.1, docs/07 §7.1 | 首页 → 座位列表 → 座位详情 → 预约提交 → 我的预约 |
| 链路二 | 管理员全流程管理（闭环路径） | docs/04 §5.1, docs/07 §7.2 | 管理员登录 → 座位管理 → 预约管理 → 统计 → 退出 |

### 1.2 为什么选这两条

| 原因 | 说明 |
|------|------|
| 覆盖全部 9 页 | 两条链路覆盖用户端 5 页 + 管理端 4 页，无页面遗漏 |
| 覆盖全部 CRUD | 含 SELECT(浏览) / INSERT(预约) / UPDATE(取消) / DELETE(无，软启用禁用) |
| 覆盖全部角色 | 链路一 = 学生角色，链路二 = 管理员角色，角色路径完整 |
| 覆盖全部三张表 | Users(查询登录) / Seats(浏览/管理) / Reservations(新增/查询/取消/统计) |
| 覆盖 Session 读写 | 账号切换(写入) / 登录校验(Session) / 退出(清除)，完整生命周期 |

### 1.3 不作为独立链路的流程

| 流程 | 归属 | 理由 |
|------|------|------|
| 学生取消预约 | 链路一的子流程 | 起始页是"我的预约页"，不涉及独立页面树路径 |
| 座位新增/编辑/切换状态 | 链路二的子操作 | 嵌入在座位管理页中，非独立页面树路径 |

## 2. 链路一：学生预约座位

### 2.1 链路全景图

```
浏览器路径：
  GET /                      → 用户首页
  GET /Home/Seats?floor=&area= → 座位列表页
  GET /Home/Detail/{id}      → 座位详情页
  GET /Home/Reserve/{seatId} → 预约提交页（表单）
  POST /Home/Reserve         → 预约提交（AJAX，JSON 返回）
  GET /Home/MyReservations   → 我的预约页（结果确认）

分支路径：
  POST /Home/Cancel          → 取消预约（链路一子流程）
  GET /Home/SwitchUser       → 切换账号（前置条件不足时）
```

### 2.2 页面与路由

| 步骤 | 页面 | 路由 | HTTP 方法 | 前置条件 |
|------|------|------|----------|---------|
| 1 | 用户首页 | `/` | GET | 无 |
| 2 | 座位列表页 | `/Home/Seats` | GET | 无 |
| 3 | 座位详情页 | `/Home/Detail/{id}` | GET | 座位 id 有效 |
| 4 | 预约提交页 | `/Home/Reserve/{seatId}` | GET | seatId 有效 + Session 有 UserId |
| 5 | 预约提交(执行) | `/Home/Reserve` | POST | Session 有 UserId |
| 6 | 我的预约页 | `/Home/MyReservations` | GET | Session 有 UserId |
| — | 取消预约(子流程) | `/Home/Cancel` | POST | Session 有 UserId + 预约记录 id |

### 2.3 Controller 动作

**HomeController**（命名空间 `LibrarySeatSystem.Controllers`）：

```csharp
public class HomeController : Controller
{
    private readonly ISeatService _seatService;
    private readonly IReservationService _reservationService;
    private readonly IStatsService _statsService;

    public HomeController(
        ISeatService seatService,
        IReservationService reservationService,
        IStatsService statsService)
    {
        _seatService = seatService;
        _reservationService = reservationService;
        _statsService = statsService;
    }

    // GET /
    public async Task<IActionResult> Index()
    {
        var stats = await _statsService.GetDashboardStatsAsync();
        var viewModel = new HomeIndexViewModel
        {
            TotalActiveSeats = stats.TotalActiveSeats,
            TodayReservations = stats.TodayReservations,
            CurrentUserName = HttpContext.Session.GetString("UserName"),
            IsLoggedIn = HttpContext.Session.GetInt32("UserId") != null
        };
        return View(viewModel);
    }

    // GET /Home/Seats
    public async Task<IActionResult> Seats(int? floor, string? area)
    {
        var seats = await _seatService.GetSeatsWithStatusAsync(floor, area);
        return View(seats);
    }

    // GET /Home/Detail/{id}
    public async Task<IActionResult> Detail(int id)
    {
        var detail = await _seatService.GetSeatDetailAsync(id);
        if (detail == null)
            return NotFound();  // 由 404 友好页面处理

        detail.IsCurrentUserLoggedIn = HttpContext.Session.GetInt32("UserId") != null;
        return View(detail);
    }

    // GET /Home/Reserve/{seatId}
    public async Task<IActionResult> Reserve(int seatId)
    {
        if (HttpContext.Session.GetInt32("UserId") == null)
            return RedirectToAction("SwitchUser");

        var seat = await _seatService.GetSeatByIdAsync(seatId);
        if (seat == null || !seat.IsActive)
            return NotFound();  // 座位不存在或已禁用

        return View(seat);
    }

    // POST /Home/Reserve
    [HttpPost]
    public async Task<IActionResult> Reserve(int seatId, DateTime date, TimeSpan startTime, TimeSpan endTime)
    {
        var userId = HttpContext.Session.GetInt32("UserId");
        if (userId == null)
            return Json(new { success = false, message = "请先选择学生账号" });

        // 后端参数校验：时段合法性
        if (startTime >= endTime)
            return Json(new { success = false, message = "结束时间必须晚于开始时间" });

        if (startTime.TotalHours < 8 || endTime.TotalHours > 22)
            return Json(new { success = false, message = "预约时段必须在 8:00-22:00 之间" });

        if (endTime - startTime > TimeSpan.FromHours(4))
            return Json(new { success = false, message = "单次预约最长4小时" });

        if (endTime - startTime < TimeSpan.FromHours(1))
            return Json(new { success = false, message = "预约时长至少 1 小时" });

        if (date < DateTime.Today)
            return Json(new { success = false, message = "不能预约过去的日期" });

        var (success, message) = await _reservationService.CreateAsync(
            userId.Value, seatId, date, startTime, endTime);

        return Json(new { success, message });
    }

    // GET /Home/MyReservations
    public async Task<IActionResult> MyReservations()
    {
        var userId = HttpContext.Session.GetInt32("UserId");
        if (userId == null)
            return RedirectToAction("SwitchUser");

        var viewModel = await _reservationService.GetUserReservationsAsync(userId.Value);
        return View(viewModel);
    }

    // POST /Home/Cancel   ← 链路一子流程
    [HttpPost]
    public async Task<IActionResult> Cancel(int id)
    {
        var userId = HttpContext.Session.GetInt32("UserId");
        if (userId == null)
            return Json(new { success = false, message = "请先选择学生账号" });

        var (success, message) = await _reservationService.CancelAsync(id, userId.Value);
        return Json(new { success, message });
    }
}
```

#### SeatService

```csharp
public interface ISeatService
{
    Task<List<SeatWithStatus>> GetSeatsWithStatusAsync(int? floor, string? area);
    Task<SeatDetailViewModel?> GetSeatDetailAsync(int id);
    Task<Seat?> GetSeatByIdAsync(int id);
}

public class SeatService : ISeatService
{
    private readonly AppDbContext _db;

    public SeatService(AppDbContext db) { _db = db; }

    public async Task<List<SeatWithStatus>> GetSeatsWithStatusAsync(int? floor, string? area)
    {
        var today = DateTime.Today;

        // 获取今日有效预约的 SeatId 集合
        var reservedSeatIds = await _db.Reservations
            .Where(r => r.Date == today && r.Status == "已预约")
            .Select(r => r.SeatId)
            .Distinct()
            .ToListAsync();

        // 查询可用座位
        var query = _db.Seats.Where(s => s.IsActive);
        if (floor.HasValue)
            query = query.Where(s => s.Floor == floor.Value);
        if (!string.IsNullOrEmpty(area))
            query = query.Where(s => s.Area == area);

        var seats = await query.OrderBy(s => s.SeatNumber).ToListAsync();

        return seats.Select(s => new SeatWithStatus
        {
            Id = s.Id,
            SeatNumber = s.SeatNumber,
            Floor = s.Floor,
            Area = s.Area,
            Description = s.Description,
            IsOccupied = reservedSeatIds.Contains(s.Id)
        }).ToList();
    }

    public async Task<SeatDetailViewModel?> GetSeatDetailAsync(int id)
    {
        var seat = await _db.Seats.FirstOrDefaultAsync(s => s.Id == id);
        if (seat == null) return null;

        var today = DateTime.Today;
        var records = await _db.Reservations
            .Where(r => r.SeatId == id && r.Date >= today)
            .OrderBy(r => r.Date)
            .ThenBy(r => r.StartTime)
            .ToListAsync();

        return new SeatDetailViewModel
        {
            Seat = seat,
            ReservationRecords = records
        };
    }

    public async Task<Seat?> GetSeatByIdAsync(int id)
    {
        return await _db.Seats.FirstOrDefaultAsync(s => s.Id == id);
    }
}
```

#### ReservationService

```csharp
public interface IReservationService
{
    Task<(bool Success, string Message)> CreateAsync(int userId, int seatId,
        DateTime date, TimeSpan startTime, TimeSpan endTime);
    Task<MyReservationsViewModel> GetUserReservationsAsync(int userId);
    Task<(bool Success, string Message)> CancelAsync(int reservationId, int userId);
}

public class ReservationService : IReservationService
{
    private readonly AppDbContext _db;

    public ReservationService(AppDbContext db) { _db = db; }

    public async Task<(bool Success, string Message)> CreateAsync(int userId, int seatId,
        DateTime date, TimeSpan startTime, TimeSpan endTime)
    {
        // 每日预约上限检测：同一用户同一日期最多 2 次有效预约
        var todayCount = await _db.Reservations.CountAsync(r =>
            r.UserId == userId && r.Date == date && r.Status == "已预约");
        if (todayCount >= 2)
            return (false, "当日预约已达上限（最多2次）");

        // 冲突检测：同一座位 + 同一日期 + 有效状态 + 时段重叠
        var hasConflict = await _db.Reservations.AnyAsync(r =>
            r.SeatId == seatId &&
            r.Date == date &&
            r.Status == "已预约" &&
            startTime < r.EndTime &&
            endTime > r.StartTime);

        if (hasConflict)
            return (false, "该时段已被预约，请选择其他时段或座位");

        // 同用户同时段检测：同一用户同一日期相同时段不可预约多个座位
        var userConflict = await _db.Reservations.AnyAsync(r =>
            r.UserId == userId &&
            r.Date == date &&
            r.Status == "已预约" &&
            startTime < r.EndTime &&
            endTime > r.StartTime);

        if (userConflict)
            return (false, "该时段您已预约其他座位");

        // 座位是否启用
        var seat = await _db.Seats.FindAsync(seatId);
        if (seat == null || !seat.IsActive)
            return (false, "该座位不可用");

        // 创建预约记录
        var reservation = new Reservation
        {
            UserId = userId,
            SeatId = seatId,
            Date = date,
            StartTime = startTime,
            EndTime = endTime,
            Status = "已预约",
            CreatedAt = DateTime.Now
        };

        _db.Reservations.Add(reservation);
        await _db.SaveChangesAsync();

        return (true, "预约成功");
    }

    public async Task<MyReservationsViewModel> GetUserReservationsAsync(int userId)
    {
        var now = DateTime.Now;
        var today = DateTime.Today;

        var records = await _db.Reservations
            .Where(r => r.UserId == userId)
            .Include(r => r.Seat)
            .OrderByDescending(r => r.CreatedAt)
            .ToListAsync();

        var items = records.Select(r => new ReservationItem
        {
            Id = r.Id,
            SeatNumber = r.Seat.SeatNumber,
            Date = r.Date,
            TimeSlot = $"{r.StartTime:hh\\:mm} - {r.EndTime:hh\\:mm}",
            Status = r.Status,
            CreatedAt = r.CreatedAt,
            // CanCancel 条件：状态为已预约 + 未过期 + 未开始
            CanCancel = r.Status == "已预约" &&
                        (r.Date > today ||
                         (r.Date == today && r.StartTime > now.TimeOfDay))
        }).ToList();

        return new MyReservationsViewModel { Items = items };
    }

    public async Task<(bool Success, string Message)> CancelAsync(int reservationId, int userId)
    {
        var reservation = await _db.Reservations.FindAsync(reservationId);
        if (reservation == null)
            return (false, "预约记录不存在");

        if (reservation.UserId != userId)
            return (false, "只能取消自己的预约");

        if (reservation.Status != "已预约")
            return (false, "只有已预约状态的记录可以取消");

        // 时间条件检查
        var now = DateTime.Now;
        var today = DateTime.Today;
        if (reservation.Date < today)
            return (false, "已过期的预约无法取消");
        if (reservation.Date == today && reservation.StartTime <= now.TimeOfDay)
            return (false, "预约已开始，无法取消");

        reservation.Status = "已取消";
        await _db.SaveChangesAsync();

        return (true, "取消成功");
    }
}
```

### 2.5 DataAccess 动作汇总

| 步骤 | 访问对象 | 操作类型 | SQL 动作 |
|------|---------|---------|---------|
| 首页 | Seats | 读 | `SELECT COUNT(*) WHERE IsActive = 1` |
| 首页 | Reservations | 读 | `SELECT COUNT(*) WHERE Date = today AND Status = N'已预约'` |
| 座位列表 | Seats | 读 | `SELECT * WHERE IsActive = 1 [AND Floor = @floor] [AND Area = @area]` |
| 座位列表 | Reservations | 读 | `SELECT SeatId WHERE Date = today AND Status = N'已预约'` |
| 座位详情 | Seats | 读 | `SELECT * WHERE Id = @id` |
| 座位详情 | Reservations | 读 | `SELECT * WHERE SeatId = @id AND Date >= today ORDER BY Date, StartTime` |
| 预约表单 GET | Seats | 读 | `SELECT * WHERE Id = @seatId` |
| 预约提交 POST | Reservations | 冲突检测读 | `SELECT 1 WHERE SeatId=@s AND Date=@d AND Status=N'已预约' AND @st < EndTime AND @et > StartTime` |
| 预约提交 POST | Seats | 存在性读 | `SELECT IsActive WHERE Id = @seatId` |
| 预约提交 POST | Reservations | INSERT | `INSERT INTO Reservations (...) VALUES (...)` |
| 我的预约 | Reservations | 读 | `SELECT * WHERE UserId = @uid ORDER BY CreatedAt DESC`（Include Seat） |
| 取消 POST | Reservations | 读 | `SELECT * WHERE Id = @rid` |
| 取消 POST | Reservations | UPDATE | `UPDATE SET Status = N'已取消' WHERE Id = @rid` |

### 2.6 数据库变化

| 步骤 | 变更类型 | 变更内容 | 事务范围 |
|------|---------|---------|---------|
| 预约提交 POST | INSERT | 新增一条 Reservation 记录（10 字段全量写入） | 单条 INSERT，EF Core 默认事务 |
| 取消 POST | UPDATE | 修改一条 Reservation.Status = "已取消" | 单条 UPDATE，EF Core 默认事务 |

**所有其他步骤为只读操作，无数据库变更。**

### 2.7 状态流转

**预约主流程：**

```
开始(无状态) ──→ 已预约  (CreateAsync 成功时写入 Status = "已预约")
     │
     └──→ 已取消  (CancelAsync 成功时 UPDATE Status = "已取消", 终态)
```

| 状态变更 | 触发动作 | 入口方法 | 前置条件 |
|---------|---------|---------|---------|
| 无 → 已预约 | 学生提交预约 | `ReservationService.CreateAsync()` | 通过时段冲突检测 + 座位可用 |
| 已预约 → 已取消 | 学生点击取消 | `ReservationService.CancelAsync()` | 属于本人 + 状态为已预约 + 未过期 + 未开始 |

**座位状态（非链路一直接修改，但影响链路一的可用性约束）：**

```
启用(IsActive=true) ←──→ 禁用(IsActive=false)
  (管理员在管理端切换，链路一中仅读取)
```

链路一中 `SeatService.GetSeatsWithStatusAsync()` 只返回 `IsActive == true` 的座位；如果座位在预约提交前被管理员禁用，`CreateAsync` 中会拦截并返回错误提示。

### 2.8 异常与边界情况

| 场景 | 检测位置 | 处理方式 | 用户看到 |
|------|---------|---------|---------|
| 未选账号访问预约页 | HomeController.Reserve(GET) | 重定向到 SwitchUser | 跳转到账号切换页 |
| 未选账号提交预约 | HomeController.Reserve(POST) | 返回 JSON error | `{success:false, message:"请先选择学生账号"}` |
| 结束时间 ≤ 开始时间 | Controller 参数校验 | 返回 JSON error | `{success:false, message:"结束时间必须晚于开始时间"}` |
| 时段超出 8:00-22:00 | Controller 参数校验 | 返回 JSON error | `{success:false, message:"预约时段必须在 8:00-22:00 之间"}` |
| 时长超过 4 小时 | Controller 参数校验 | 返回 JSON error | `{success:false, message:"单次预约最长4小时"}` |
| 时长不足 1 小时 | Controller 参数校验 | 返回 JSON error | `{success:false, message:"预约时长至少 1 小时"}` |
| 预约过去日期 | Controller 参数校验 | 返回 JSON error | `{success:false, message:"不能预约过去的日期"}` |
| 每日预约超过 2 次 | Service.CreateAsync | 返回 (false, 提示) | `{success:false, message:"当日预约已达上限（最多2次）"}` |
| 同用户同时段冲突 | Service.CreateAsync | 返回 (false, 提示) | `{success:false, message:"该时段您已预约其他座位"}` |
| 时段冲突 | Service.CreateAsync | 返回 (false, 提示) | `{success:false, message:"该时段已被预约，请选择其他时段或座位"}` |
| 座位已禁用 | Service.CreateAsync | 返回 (false, 提示) | `{success:false, message:"该座位不可用"}` |
| 座位 id 不存在 | SeatService.GetSeatDetailAsync | 返回 null → Controller NotFound | 404 友好页面 |
| 无效座位 id | SeatService.GetSeatByIdAsync | 返回 null → Controller NotFound | 404 友好页面 |
| 取消别人的预约 | Service.CancelAsync | 返回 (false, 提示) | `{success:false, message:"只能取消自己的预约"}` |
| 取消非"已预约"记录 | Service.CancelAsync | 返回 (false, 提示) | `{success:false, message:"只有已预约状态的记录可以取消"}` |
| 取消过期预约 | Service.CancelAsync | 不可取消判断在前端 + 后端双重 | `{success:false, message:"已过期的预约无法取消"}` |
| 取消已开始预约 | Service.CancelAsync | 不可取消判断在前端 + 后端双重 | `{success:false, message:"预约已开始，无法取消"}` |
| 数据库连接失败 | Service / EF Core | 冒泡异常，页面显示"数据加载失败" | 500 错误或友好提示 |

**前端侧交互（前序文档未明确，本文补足）：**

| 交互点 | 实现方式 |
|--------|---------|
| 预约表单提交 | jQuery AJAX POST /Home/Reserve，不整页刷新 |
| 预约成功反馈 | 弹窗或 toast 提示"预约成功" + "查看我的预约"按钮 |
| 预约失败反馈 | 弹窗或 toast 显示错误信息，表单保持原数据 |
| 取消确认 | 弹窗 `confirm("确定取消该预约？")`，确认后 AJAX 提交 |
| 取消成功反馈 | toast 提示"取消成功"，自动刷新列表 |
| 不可取消条件 | 前端判断取消按钮 disabled 或隐藏（条件见 §2.7 的 CanCancel 计算） |

## 3. 链路二：管理员全流程管理

### 3.1 链路全景图

```
浏览器路径：
  GET /Admin/Login            → 管理员登录页
  POST /Admin/Login           → 登录校验 → 重定向
  GET /Admin/Seats            → 座位管理页（登录后默认页）
  GET /Admin/SeatCreate       → 新增座位表单
  POST /Admin/SeatCreate      → 提交保存 → 重定向
  GET /Admin/SeatEdit/{id}    → 编辑座位表单
  POST /Admin/SeatEdit        → 提交保存 → 重定向
  POST /Admin/SeatToggle      → 启用/禁用（AJAX）
  GET /Admin/Reservations     → 预约管理页（筛选）
  GET /Admin/Stats            → 统计页
  ← 任意后台页 → "退出"       → Session 清除 → 重定向登录页
```

### 3.2 页面与路由

```csharp
public class AdminController : Controller
{

    public IActionResult Login()
    {
        // 如果已登录，直接跳转座位管理页
        if (HttpContext.Session.GetInt32("AdminId") != null)
            return RedirectToAction("Seats");
        return View();
    }

    // POST /Admin/Login
    [HttpPost]
    [TypeFilter(typeof(AdminAuthFilter), Order = -1)]
    public async Task<IActionResult> Login(string username)
    {

        {

        }

            return View();
        }

        HttpContext.Session.SetInt32("AdminId", user.Id);
        HttpContext.Session.SetString("AdminName", user.DisplayName);
        return RedirectToAction("Seats");
    }

    // POST /Admin/Logout
    [HttpPost]
    public IActionResult Logout()
    {
        HttpContext.Session.Remove("AdminId");
        HttpContext.Session.Remove("AdminName");
        return RedirectToAction("Login");
    }

    /* ---------- 座位管理 ---------- */

    // GET /Admin/Seats
    public async Task<IActionResult> Seats()
    {
        var seats = await _seatService.GetAllSeatsAsync();
        return View(seats);
    }

    // GET /Admin/SeatCreate
    public IActionResult SeatCreate()
    {
        return View(new Seat());
    }

    // POST /Admin/SeatCreate
    [HttpPost]
    public async Task<IActionResult> SeatCreate(Seat model)
    {
        if (!ModelState.IsValid)
            return View(model);

        var result = await _seatService.CreateSeatAsync(model);
        if (!result.Success)
        {
            ModelState.AddModelError("", result.Message);
            return View(model);
        }

        return RedirectToAction("Seats");
    }

    {
        var seat = await _seatService.GetSeatByIdAsync(id);
        if (seat == null) return NotFound();
        return View(seat);
    }

    // POST /Admin/SeatEdit

    public async Task<IActionResult> SeatEdit(Seat model)
    {

            return View(model);

        var result = await _seatService.UpdateSeatAsync(model);
        if (!result.Success)
        {
            ModelState.AddModelError("", result.Message);
            return View(model);
        }

        if (seat == null)
```

```csharp
public interface ISeatService
{
    // …已有方法（见链路一）

    // 以下为管理端追加
    Task<List<Seat>> GetAllSeatsAsync();
    Task<(bool Success, string Message)> CreateSeatAsync(Seat seat);
    Task<(bool Success, string Message)> UpdateSeatAsync(Seat seat);
    Task SaveChangesAsync();
    Task<User?> ValidateAdminAsync(string username);  // 移到 UserService 更合理
}

public class SeatService : ISeatService
{
    // …已有方法

    public async Task<List<Seat>> GetAllSeatsAsync()
    {
        return await _db.Seats.OrderBy(s => s.SeatNumber).ToListAsync();
    }

    public async Task<(bool Success, string Message)> CreateSeatAsync(Seat seat)
    {
        // 检查座位编号是否重复
        var exists = await _db.Seats.AnyAsync(s => s.SeatNumber == seat.SeatNumber);
        if (exists)
            return (false, "座位编号已存在");

        _db.Seats.Add(seat);
        await _db.SaveChangesAsync();
        return (true, "新增成功");
    }

    public async Task<(bool Success, string Message)> UpdateSeatAsync(Seat seat)
    {
        var existing = await _db.Seats.FindAsync(seat.Id);
        if (existing == null)
            return (false, "座位不存在");

        // 检查编号是否与其他座位冲突（排除自身）
        var duplicate = await _db.Seats.AnyAsync(s =>
            s.SeatNumber == seat.SeatNumber && s.Id != seat.Id);
        if (duplicate)
            return (false, "座位编号已被其他座位使用");

        existing.SeatNumber = seat.SeatNumber;
        existing.Floor = seat.Floor;
        existing.Area = seat.Area;
        existing.Description = seat.Description;

        await _db.SaveChangesAsync();
        return (true, "更新成功");
    }

    public async Task SaveChangesAsync()
    {
        await _db.SaveChangesAsync();
    }

    public async Task<User?> ValidateAdminAsync(string username)
    {
        return await _db.Users.FirstOrDefaultAsync(u =>
            u.Username == username && u.IsAdmin);
    }
}
```

```csharp
        var query = _db.Reservations
            .Include(r => r.User)
            .Include(r => r.Seat)
            .AsQueryable();

        if (!string.IsNullOrEmpty(status))
            query = query.Where(r => r.Status == status);
        if (dateFrom.HasValue)
            query = query.Where(r => r.Date >= dateFrom.Value);
        if (dateTo.HasValue)
            query = query.Where(r => r.Date <= dateTo.Value);

        var records = await query.OrderByDescending(r => r.CreatedAt).ToListAsync();

        return new AdminReservationsViewModel
        {
            Items = records.Select(r => new AdminReservationItem
            {
                Id = r.Id,
                UserName = r.User.DisplayName,
                SeatNumber = r.Seat.SeatNumber,
                Date = r.Date,
                TimeSlot = $"{r.StartTime:hh\\:mm} - {r.EndTime:hh\\:mm}",
                Status = r.Status,
                CreatedAt = r.CreatedAt
            }).ToList()
```

```csharp
public interface IStatsService
{
    Task<DashboardStats> GetDashboardStatsAsync();    // 链路一
    Task<AdminStats> GetAllStatsAsync();              // 链路二追加
}

public class StatsService : IStatsService
{
    private readonly AppDbContext _db;

    public StatsService(AppDbContext db) { _db = db; }

    public async Task<DashboardStats> GetDashboardStatsAsync()
    {
        var today = DateTime.Today;
        return new DashboardStats
        {
            TotalActiveSeats = await _db.Seats.CountAsync(s => s.IsActive),
            TodayReservations = await _db.Reservations
                .CountAsync(r => r.Date == today && r.Status == "已预约")
        };
    }

    public async Task<AdminStats> GetAllStatsAsync()
    {
        var today = DateTime.Today;
        var fourteenDaysAgo = today.AddDays(-13);

        // 基础计数
        var totalActiveSeats = await _db.Seats.CountAsync(s => s.IsActive);
        var totalReservations = await _db.Reservations.CountAsync();
        var todayReservations = await _db.Reservations
            .CountAsync(r => r.Date == today);

        // 座位热度排行（只计有效预约）
        var seatRankings = await _db.Reservations
            .Where(r => r.Status == "已预约")
            .GroupBy(r => r.SeatId)
            .Select(g => new
            {
                SeatId = g.Key,
                Count = g.Count()
            })
            .OrderByDescending(x => x.Count)
            .Take(10)
            .ToListAsync();

        var rankingViewModels = seatRankings.Select(x => new SeatRanking
        {
            SeatNumber = _db.Seats.Find(x.SeatId)?.SeatNumber ?? "未知",
            Count = x.Count
        }).ToList();

        // 近 14 日趋势
        var dailyTrends = await _db.Reservations
            .Where(r => r.Date >= fourteenDaysAgo && r.Date <= today)
            .GroupBy(r => r.Date)
            .Select(g => new DailyTrend
            {
                Date = g.Key,
                Count = g.Count()
            })
            .OrderBy(t => t.Date)
            .ToListAsync();

        return new AdminStats
        {
            TotalActiveSeats = totalActiveSeats,
            TotalReservations = totalReservations,
            TodayReservations = todayReservations,
            SeatRankings = rankingViewModels,
            DailyTrends = dailyTrends
        };
    }
}
```

```csharp
// AdminAuthFilter 工作原理
// 适用范围：AdminController 上标记 [ServiceFilter(typeof(AdminAuthFilter))]

                StatusCode = 401

        }
    }
```

### 5.3 设计决策记录

| 决策 | 选项 | 选择 | 理由 |
|------|------|------|------|
| 取消请求用 POST 还是 DELETE | POST / DELETE | POST | Bootstrap 表单只支持 GET/POST，不使用 JavaScript 发送 DELETE 方法 |
| 座位创建/编辑成功后跳转 | 停留在当前页 / 重定向到列表 | 重定向到列表 | 防止表单重复提交 |
| 统计页数据用 ViewBag 还是 ViewModel | ViewBag / ViewModel | ViewModel | 类型安全，ViewModel 结构已在 docs/08 定义 |
| 预约提交用表单提交还是 AJAX | 表单提交 / AJAX | AJAX | 不整页刷新，在同一页显示成功/失败提示 |
| 管理端退出用 GET 还是 POST | GET / POST | POST | 防止 CSRF，符合 REST 规范 |
| AdminAuthFilter 作用范围 | Controller 级别 / Action 级别 | Controller 级别 | Login 和 Logout 通过 Order 和显式标记绕过，减少重复代码 |
