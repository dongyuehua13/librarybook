### 2. 管理员登录是否生效？

**✅ 生效。** 验证链路：
- GET `/Admin/Login` → 显示用户名输入框 ✅
- POST 提交 "admin" → `UserService.ValidateAdminAsync("admin")` → 写入 Session (`AdminId`, `AdminName`) → 重定向 `/Admin/Seats` ✅
- POST 提交不存在用户名 → ViewBag.Error = "管理员账号不存在" → 返回表单 ✅
- 已登录再次访问 `/Admin/Login` → `Session["AdminId"]` 存在 → 自动跳转 `/Admin/Seats` ✅
- Logout → 清除 Session → 重定向 Login ✅
- `AdminAuthFilter` 拦截所有非 Login/Logout 的 Action ✅

### 5. 统计页是否真的基于数据库数据？

**✅ 基于数据库实时查询。** `StatsService.GetAllStatsAsync()` 每项均为 EF Core 查询：
- `TotalActiveSeats` = `_db.Seats.CountAsync(s => s.IsActive)` ✅
- `TotalReservations` = `_db.Reservations.CountAsync()` ✅
- `TodayReservations` = `_db.Reservations.CountAsync(r => r.Date == today && r.Status == "已预约")` ✅
- `SeatRankings` = `GROUP BY SeatId WHERE Status=已预约 ORDER BY COUNT DESC TOP 10` ✅
- `DailyTrends` = `GROUP BY Date WHERE Date >= 14日前` ✅
- 无硬编码数据 ✅
